                                    # 2. 非递归的预测分析法

                                    非递归预测分析不需要为每个非终结符编写递归下降过程，根据预测分析表构造自动机，也叫表驱动的预测分析。

                                    # 2.1 文法转换

                                    左递归文法会使递归下降分析器陷入无限循环。
                                    直接左递归的文法:

                                    A→AαA{\rightarrow} A{\alpha} A→Aα

                                    经过两步或两步以上的推导产生的左递归是间接左递归的。

                                    # 消除直接左递归:

                                    A→Aα∣β(a≠ϵ,β不以A开头)A{\rightarrow}A{\alpha}|{\beta}(a\not=\epsilon,{\beta}\text{不以A开头}) A→Aα∣β(a=ϵ,β不以A开头)

                                    替换为:

                                    A→βA′A{\rightarrow}{\beta}A^{'} A→βA′

                                    A′→αA′∣ϵA^{'}{\rightarrow}{\alpha}A^{'}|\epsilon A′→αA′∣ϵ

                                    消除左递归的一般形式:

                                    A→Aa1∣Aa2∣...∣Aa3∣β1∣β2∣...∣βn∣(ai≠ϵ,βj不以A开头)A{\rightarrow}Aa_{1}|Aa_{2}|...|Aa_{3}|{\beta}_{1}|{\beta}_{2}|...|{\beta}_{n}|(a_{i}\not=\epsilon,{\beta_{j}}\text{不以A开头}) A→Aa1​∣Aa2​∣...∣Aa3​∣β1​∣β2​∣...∣βn​∣(ai​=ϵ,βj​不以A开头)

                                    替换为:

                                    A→β1A′∣β2A′∣...∣β3A′A{\rightarrow}{\beta}_{1}A^{'}|{\beta}_{2}A^{'}|...|{\beta}_{3}A^{'} A→β1​A′∣β2​A′∣...∣β3​A′

                                    A′→a1A′∣a2A′∣...∣anA′∣ϵA^{'}{\rightarrow}a_{1}A^{'}|a_{2}A^{'}|...|a_{n}A^{'}|\epsilon A′→a1​A′∣a2​A′∣...∣an​A′∣ϵ

                                    # 消除间接左递归

                                    S→Aa∣bS{\rightarrow}Aa|b S→Aa∣b

                                    A→Ac∣Sd∣ϵA{\rightarrow}Ac|Sd|\epsilon A→Ac∣Sd∣ϵ

                                    此时将SSS的定义代入A−A-A−产生式，得:

                                    A→Ac∣Aad∣bd∣ϵA{\rightarrow}Ac|Aad|bd|\epsilon A→Ac∣Aad∣bd∣ϵ

                                    消除A−A-A−产生式的直接左递归:

                                    A→bdA′∣A′A{\rightarrow}bdA^{'}|A^{'} A→bdA′∣A′

                                    A′→cA′∣adA′∣ϵA^{'}{\rightarrow}cA^{'}|adA^{'}|\epsilon A′→cA′∣adA′∣ϵ

                                    # 2.2 FIRST集、SELECT集、FOLLOW集

                                    FIRSTFIRSTFIRST集的对象是串。可以包含ϵ\epsilonϵ。
                                    FOLLOWFOLLOWFOLLOW集的对象是非终结符。不可以包含ϵ\epsilonϵ。
                                    SELECTSELECTSELECT集的对象是产生式。不可以包含ϵ\epsilonϵ。

                                    1. FIRSTFIRSTFIRST集

                                      α{\alpha}α的串首终结符集FIRST(α)FIRST({\alpha})FIRST(α)集的定义为:
                                      可以从α{\alpha}α推导出来的所有串首终结符构成的集合。 若α→ϵ{\alpha}{\rightarrow}\epsilonα→ϵ，那么ϵ\epsilonϵ也在FIRST(α)FIRST({\alpha})FIRST(α)中。

                                    2. SELECTSELECTSELECT集

                                      若ϵ∈FIRST(α){\epsilon}{\in}FIRST({\alpha})ϵ∈FIRST(α)， 那么SELECT(A→α)=(FIRST(α)−{ϵ})∪FOLLOW(A)SELECT(A{\rightarrow}{\alpha})=(FIRST({\alpha})-{\{}{\epsilon}{\}}){\cup}FOLLOW(A)SELECT(A→α)=(FIRST(α)−{ϵ})∪FOLLOW(A)

                                    3. FOLLOWFOLLOWFOLLOW集 个人理解:FOLLOW(A)FOLLOW(A)FOLLOW(A)为由A产生的句型中紧跟在A后面的终结符。
                                      定义:

                                    算法:

                                    1. 将$\$$放入FOLLOW(S)FOLLOW(S)FOLLOW(S)中，其中SSS为开始符号，$\$$为输入右端的结束标记。
                                    2. 如果存在一个产生式A→αBβA{\rightarrow}{\alpha}B{\beta}A→αBβ，那么FIRST(β)FIRST({\beta})FIRST(β)中除了ϵ\epsilonϵ之外的所有符号都在FOLLOW(B)FOLLOW(B)FOLLOW(B)中。
                                    3. 如果存在一个产生式A→αBA{\rightarrow}{\alpha}BA→αB，或存在产生式A→αBβA{\rightarrow}{\alpha}B{\beta}A→αBβ，且FIRST(β)FIRST({\beta})FIRST(β)包含ϵ\epsilonϵ，那么FOLLOW(A)中的所有符号都在FOLLOW(B)FOLLOW(B)FOLLOW(B)中。

                                    # 2.3 LL(1)文法

                                    简洁定义:

                                    ∀A→α1∣α2∣...∣αn{\forall}A{\rightarrow}{\alpha}_{1}|{\alpha}_{2}|...|{\alpha}_{n} ∀A→α1​∣α2​∣...∣αn​

                                    SELECT(A→α1)∩SELECT(A→α2)∩...∩SELECT(A→αn)==∅SELECT(A{\rightarrow}{\alpha}_{1}){\cap}SELECT(A{\rightarrow}{\alpha}_{2}){\cap}...{\cap}SELECT(A{\rightarrow}{\alpha}_{n})==\emptyset SELECT(A→α1​)∩SELECT(A→α2​)∩...∩SELECT(A→αn​)==∅

                                    文法G是LL(1)的，当且仅当G的任意两个具有相同左部的产生式A→α∣βA{\rightarrow}{\alpha}|{\beta}A→α∣β满足以下条件 (同一非终结符的各个产生式的可选集互不相交):

                                    1. 不存在终结符aaa使得α{\alpha}α和β{\beta}β都能够推导出以aaa开头的串。
                                    2. α{\alpha}α和β{\beta}β至多有一个能推导出ϵ{\epsilon}ϵ。
                                    3. 若β→ϵ{\beta}{\rightarrow}\epsilonβ→ϵ，则FIRST(α)∩FOLLOW(A)=∅FIRST(\alpha){\cap}FOLLOW(A)=\emptysetFIRST(α)∩FOLLOW(A)=∅
                                      若α→ϵ{\alpha}{\rightarrow}\epsilonα→ϵ，则FIRST(β)∩FOLLOW(A)=∅FIRST(\beta){\cap}FOLLOW(A)=\emptysetFIRST(β)∩FOLLOW(A)=∅
